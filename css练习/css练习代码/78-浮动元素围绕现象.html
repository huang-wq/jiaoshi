缪毅（左一）组织召开村民小组会“做事和做人是一样的，要用真情。 你对他一分好，他会回报你十分好，淳朴的百姓都是记恩的。面对一个个问题， 我时常会想起父母说过的这句话，在日常工作中，我用自己的真情不断感化百姓。　　 每次走进村民家中，我都是先聊家常，从聊家常中逐渐了解家庭情况，时不时也向村民讲解一些扶贫政策。 这样一来，不仅村民更容易接受政策的宣传，也拉近了我们之间的距离，工作也就更容易开展了。　　 还记得在我包保的网格内召开第一次村民小组会时，百姓的问题都指向一点： 组里缺少3条串寨路（机耕道）。一个组的亲戚串寨都是走泥路，非常不便，而且其间还有很多田土， 耕种也很困难。为此，在会后我便同百姓一起徒步勘测了这3条路，并上报了市委办。 一路随行的88岁老大爷许连昌对我说：“小缪，你能为我们九组修路，方便寨邻，路要过谁家的田土， 大家都会义务拿出来，不要一分钱。非常感谢你，希望能在我有生之年看到路通。”通过各方努力， 这3条路已经硬化了1条，另外2条已经通了毛坯路。 缪毅（左一）组织召开村民小组会“做事和做人是一样的，要用真情。 你对他一分好，他会回报你十分好，淳朴的百姓都是记恩的。面对一个个问题， 我时常会想起父母说过的这句话，在日常工作中，我用自己的真情不断感化百姓。
